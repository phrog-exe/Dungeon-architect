# Dungeon-architect
Dungeon Architect to narzędzie desktopowe służące do proceduralnego generowania układów lochów, które mogą zostać wykorzystane w grach typu roguelike lub sesjach RPG. Aplikacja pozwala na szybkie tworzenie unikalnych map o zadanych parametrach, łącząc logikę algorytmiczną z nowoczesnym interfejsem graficznym.
